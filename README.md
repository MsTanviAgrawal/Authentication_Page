# Authentication Page

This project is a simple and clean authentication system that includes both **Login** and **Signup** functionality. It is designed to provide users with a smooth experience while also helping me practice frontend and backend authentication concepts.

## Overview

The Authentication Page allows users to:

* Create a new account (Signup)
* Log into an existing account
* Validate user input
* Handle authentication smoothly and securely

This project is built to understand the basics of user authentication, form handling, and connecting frontend to backend APIs.

## Features

* Simple Login & Signup forms
* Input validation
* Clean and user-friendly UI
* Backend API for authentication
* Secure user storage using a database

## Tech Stack

### **Frontend**

* React JS
* JavaScript
* CSS / Tailwind / Custom Styling

### **Backend**

* Node.js
* Express.js

### **Database**

* MongoDB (Mongoose)

## Running the Project Locally

### **1. Clone the repository:**

```bash
git clone https://github.com/MsTanviAgrawal/Authentication_Page.git
```

### **2. Frontend Setup:**

```bash
cd frontend
npm install
npm start
```

Visit: **[http://localhost:3000/](http://localhost:3000/)**

### **3. Backend Setup:**

```bash
cd backend
npm install
npm start
```

Backend runs on **[http://localhost:5000/](http://localhost:5000/)**
